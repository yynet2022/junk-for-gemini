## 指示
指定したジャンルで検索する際に役立つ、関連キーワードを2つのカテゴリに分けて10個ずつ提案してください。
- （ほぼ）普遍的に関連する語句: 指定したジャンルのニュースや学習において、時代を問わず基本となる重要なキーワード。
- 「今」を反映する関連語句: 指定したジャンルでの現在の情勢や政策、トレンドを反映した、今まさに注目されているキーワード。

## 出力形式
- 普遍の語句と旬の語句を、以下の JSON 形式で出力してください。
- "term"の **語句** は、Google検索でそのまま使えるような **シンプルな単語や略称** にしてください。（例: "GDP", "GX"）
- "description" の **簡易説明** に、その語句の簡易的な説明（必要であれば正式名称も含む）を入れてください。

{
  "universal_terms": [
    {"term": "語句1", "description": "簡易説明"},
    {"term": "語句2", "description": "簡易説明"},
    ...
  ],
  "current_terms": [
    {"term": "語句A", "description": "簡易説明"},
    {"term": "語句B", "description": "簡易説明"},
    ...
  ]
}
